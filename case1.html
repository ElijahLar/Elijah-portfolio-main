Coffee Machine - Done Right

A coffee-machine interface redesign centered on guidance, feedback, and low-friction choices

This project explored how a coffee-machine interface could feel more intuitive and reassuring by improving visibility, reducing ambiguity, and making key actions easier to recognize. The goal was to simplify the decision flow without stripping away useful control.

The Design Challenge

Coffee machines often compress a lot of information into a very small surface. If the interface is too sparse, users have to guess what happens next. If it is too dense, they hesitate. This redesign focused on balancing clarity and simplicity so the machine feels approachable even for first-time users.

Reduce uncertainty

The interface needed clearer action points so users could tell what was tappable, what step they were in, and how to undo a choice if needed.

Support recognition over recall

Options had to be visually understandable at a glance rather than relying on prior familiarity with the machine.

Keep the visual language calm

The layout aimed to feel modern and focused without overwhelming the user with unnecessary decoration.

Sketches and Interaction Approaches

The early design work explored multiple interaction patterns to test how direct the interface should feel. Each concept tried a different balance between explicit controls and minimalist presentation.

  • Sketch 1: Used visible buttons and direct touch targets to make the next action obvious.
  • Sketch 2: Reduced the amount of visible chrome while still preserving orientation and clarity.
  • Sketch 3: Leaned more on familiarity and conversational cues, which made it elegant but less immediately self-explanatory.

Design Decisions

The final direction focused on the interaction principles that made the machine easiest to understand quickly.

  • Visibility of system status: The interface communicates choices and progress clearly so users know where they are in the flow.
  • User control and freedom: Essential actions such as canceling or going back remain available and visible.
  • Aesthetic and minimalist design: The layout removes noise and lets the primary choice stand out.
  • Consistency and standards: Typography, spacing, and layout patterns stay predictable throughout the interaction.

Outcome

This redesign demonstrates how even a compact device interface benefits from strong affordances, clearer feedback, and less guesswork. The solution moved the experience closer to Nielsen's heuristics by helping users recognize actions faster, recover more easily, and trust the machine's behavior.
